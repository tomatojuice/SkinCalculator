# 🎨 着せ替え電卓 (Skin Calculator)

[English](README.md) | 日本語

スタイリッシュで多機能、自分好みに「着せ替え」ができる電卓アプリです。
日々の計算を、もっと楽しく、もっと便利に。

---

## 🌟 特徴

- **🎨 豊富なテーマ:** ダーク、ライト、ピンクなど、気分に合わせてスキンを自由に変更可能。
- **💰 消費税計算:** 設定画面から税率をカスタマイズして、ボタン一つで税込・税抜計算。
- **📝 計算履歴:** 過去の計算結果をいつでもリストで確認できます。
- **🌐 バイリンガル:** 日本語と英語を完全にサポート。
- **📳 快適な操作感:** キー入力時のバイブレーションフィードバック（ON/OFF可能）。
